## 🧠 2. Descripción del proyecto

Este proyecto consiste en una landing page profesional tipo portafolio digital que presenta el perfil, habilidades y áreas de especialidad de un productor audiovisual en proceso de formación como ingeniero multimedia.

### 🎯 ¿Qué hace el proyecto?
Permite visualizar de forma clara, estética y profesional la información personal, experiencia, áreas de especialización y medios de contacto.

### ⚙️ ¿Qué problema soluciona?
Soluciona la necesidad de tener una presencia digital profesional centralizada que comunique valor creativo y técnico en un solo lugar.

### 👤 ¿Para quién está pensado?
- Reclutadores  
- Empresas del sector creativo y tecnológico  
- Clientes potenciales  
- Colaboradores profesionales

## ⚙️ 5. Funcionalidades

- Navegación interactiva
- Menú superior con secciones internas y dropdown.
- Sección Hero (presentación)
- Imagen de perfil, nombre y llamado a la acción.
- Áreas de especialidad
- Botones interactivos con efecto hover reveal (muestran información adicional).
- Formulario de contacto
- Permite ingresar datos para propuestas de colaboración.
- Diseño responsive
- Adaptable a dispositivos móviles, tablet y escritorio.
- Microinteracciones
- Animaciones suaves en hover (botones, cards, navegación).
